# Paint3D
A program allowing painting textures of different channels SIMULTANEOUSLY on a 3d model.&#10
You can also see this page: http://www.cnblogs.com/dydx/p/6445845.html

###Preliminary
Qt 4.7 or 4.8 32bit
OpenGL 32bit
